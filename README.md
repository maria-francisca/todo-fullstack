# To-Do Fullstack

Este é um projeto de listar tarefas com:
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express + MongoDB 


## Como correr o programa

### Pré-requisitos

- Node.js e npm instalados
- MongoDB em execução (para o backend)

### Backend (Swagger)

````
cd backend
npm install
node src/server.js
````

O backend estará disponível em http://localhost:5000.

A documentação da API (Swagger) estará em http://localhost:5000/api-docs/


### Frontend
````
npm create vite@latest
# Selecionar: React + JavaScript
cd frontend
npm install
npm install tailwindcss @tailwindcss/vite
npm run dev
````

O frontend está disponível em http://localhost:5173/

## Funcionalidades

- Criar tarefas
- Listar tarefas
- Marcar tarefas como completas
- Apagar tarefas
